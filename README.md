## 🧠 Présentation

Un jeu de survie dynamique où vous incarnez un astronaute luttant pour maintenir son vaisseau spatial opérationnel après avoir subi de lourds dégâts en traversant les dangers de l’espace.

Le joueur doit identifier rapidement les pannes, intervenir efficacement et gérer une pression croissante à mesure que de nouveaux problèmes apparaissent.

L’objectif est de survivre le plus longtemps possible.

Ce projet a été développé en 48 heures dans le cadre de la Global Game Jam 2020 à Genève, sur le thème « Repair ».
